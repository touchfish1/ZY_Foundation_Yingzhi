## 1. 项目概述

- **项目名称**：Project ZHANGYUAN（ZY_Foundation_Yingzhi）
- **项目定位**：AI 模型 API 服务平台，聚合主流大模型能力，提供统一 API 接入、管理后台和开发者工具
- **技术栈**：
  - 后端：Java 21 + Spring Boot 3.3.5 + Gradle + Spring Cloud Gateway + Nacos
  - 前端（对外网站）：Nuxt 3 + Vue 3 + TypeScript
  - 前端（管理后台）：Vue 3 + Vite 6 + Naive UI + Pinia
  - 数据库：PostgreSQL + Redis + MinIO
  - 部署：Docker Compose + Kubernetes + Helm
  - CI/CD：GitHub Actions
